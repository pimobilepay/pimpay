# 🏦 PimPay: Empowering Financial Freedom for All (v1.0)

## 🌟 Vision du Projet
PimPay est une solution de banque virtuelle conforme à la Charia. L'architecture repose sur Next.js et TypeScript pour garantir une sécurité maximale et une gestion rigoureuse des types de données financières.

## 📂 Structure du Projet
* **app/** : Routes et pages principales.
* **components/** : Composants UI (.tsx) comme la WalletPage.
* **context/** : Gestion de l'état (Auth, Wallet).
* **services/** : Logique métier et conformité (compliance.ts).
* **lib/** : Utilitaires et géo-restriction.

## 🛡️ Sécurité & Confidentialité
* **Masquage Dynamique** : Protection des données sensibles à l'écran.
* **Validation TypeScript** : Typage strict pour éviter les erreurs de calcul.
* **Frais Sidra Chain** : Frais indexés sur les coûts réels et minimes du réseau (Anti-Gharar).

## ⚖️ Conformité Réglementaire & Juridique (Réponse Basel III)
Pour mitiger les risques liés aux licences bancaires identifiés dans le rapport :
* **Approche par Partenariat (BaaS)** : Utilisation de licences d'institutions financières établies pour opérer légalement.
* **Standard Basel III** : Modules de reporting pour le suivi des ratios de liquidité.
* **Géo-blocage** : Restriction par IP pour limiter l'accès aux juridictions autorisées.

## 📜 Engagement de Qualité
Chaque ligne de code de PimPay est typé pour supprimer toute complexité inutile et garantir la sécurité absolue des fonds.
